Technology Stack

Frontend:
Next.js (React-based framework for server-side rendering and optimized performance).
GraphQL Apollo(for querying data, providing efficient and flexible APIs).
Tailwind CSS (for responsive, utility-first styling).

Backend:
NestJS (TypeScript-based backend framework for building scalable, modular APIs).
GraphQL (for seamless data communication between frontend and backend).
MongoDB (NoSQL database for flexibility and scalability with documents).

Authentication:
JWT (JSON Web Tokens) for authentication and user authorization.
Send Token via request headers

Payment Handling:
Virtual Credits (no real-world payments in MVP; users receive a 50-credit balance upon registration to purchase tickets).

Later: Group Chats via WebSockets
